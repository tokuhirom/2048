# 2048

スマホブラウザで快適に遊べる2048ゲームです。PWA対応でホーム画面に追加してオフラインでもプレイできます。

## 遊び方

- スワイプ（スマホ）または矢印キー（PC）でタイルを移動
- 同じ数字のタイルがぶつかると合体して倍の数字になる
- 2048のタイルを作ればクリア

## 機能

- タッチスワイプ操作
- レスポンシブデザイン
- スコア・ベストスコア保存
- Undo（1手戻し）
- PWA（オフライン対応）

## デプロイ

GitHub Pages で公開できます。Settings > Pages から `main` ブランチを選択してください。
